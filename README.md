# LanForge

**LanForge** is an offline LAN multiplayer gaming platform that enables multiple devices on the same local network to create rooms, host game sessions, and play multiplayer games without requiring an internet connection.

LanForge focuses on building a robust core system for local multiplayer experiences, where games can be plugged in as modules and shared seamlessly across connected devices.

---

## Why LanForge?
- No internet required
- Local multiplayer over Wi-Fi or hotspot
- Ideal for colleges, hostels, events, and travel
- Low-latency LAN gameplay
- Foundation for multiple casual and competitive games

---

## Core Concepts
- **Rooms** for grouping players on a local network
- **Host-based game sessions** with controlled authority
- **Role management** (host, leaders, players, spectators)
- **Modular game architecture** (games as plugins)
- **State-driven multiplayer synchronization**

---

## Planned Features
- Room creation and discovery on local network
- Player join/leave handling with host migration
- Multiple game sessions per room
- Spectator support
- Modular game system for adding new games
- Offline-first design with optional update sync when internet is available

---

## Long-Term Vision
- Expandable library of local multiplayer games
- Developer-friendly framework for adding new games
- Reliable offline gameplay with deterministic synchronization
